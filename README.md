# ContadorSwing

Aplicação desktop desenvolvida em Java Swing.

## Funcionalidades

* Definição do valor inicial
* Definição do valor final
* Definição do passo
* Exibição da sequência em uma lista
* Interface gráfica utilizando Swing

## Tecnologias

* Java 21
* Swing
* Maven

## Como executar

### Pelo NetBeans

Abra o projeto e execute a classe principal:

`TelaSwing.java`

### Pelo arquivo JAR

```bash
java -jar ContadorSwing-1.0-SNAPSHOT.jar
```
